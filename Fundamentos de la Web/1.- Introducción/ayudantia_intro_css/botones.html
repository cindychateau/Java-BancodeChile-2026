Práctica de Botones CSS

En este ejercicio aprenderemos a cambiar el tamaño, el borde y la sombra de los botones.

Botón Buscar

Este botón usa border para el borde y border-radius para las esquinas redondeadas.

Botón + (Agregar)

Este botón usa border-radius: 50% para convertirse en un círculo. El alto y el ancho deben ser iguales.

Botón Crear Cuenta

Este botón usa box-shadow para agregar una sombra y darle profundidad.
