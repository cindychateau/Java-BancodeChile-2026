Selectores CSS

Aprenderemos a apuntar a elementos específicos con distintos tipos de selectores.

1. Selector de Etiqueta

Se escribe con el nombre de la etiqueta. Aplica a TODOS los elementos de ese tipo.

Ejemplo: p { color: blue; }

Todos estos párrafos reciben su estilo por el selector de etiqueta p.

  • Este li recibe su estilo por el selector de etiqueta li
  • Este también, sin necesidad de clases ni ids
  • Todos los li de la página reciben el mismo estilo

2. Selector de Clase

Se escribe con un punto (.) antes del nombre. Se puede usar en varios elementos.

Ejemplo: .resaltado { background-color: yellow; }

Este párrafo es normal, sin ninguna clase.

Este párrafo tiene class="resaltado": fondo amarillo.

Este párrafo tiene class="importante": texto en rojo.

Este párrafo tiene dos clases a la vez: combina ambos estilos.

Este div tiene class="caja-clase".
Este otro div reutiliza la misma clase.

3. Selector de ID

Se escribe con un # antes del nombre. El ID debe ser único, no puede repetirse en la página.

Ejemplo: #caja-unica { background-color: pink; }

Soy el único elemento con id="caja-unica". Mi estilo es exclusivo.

4. Selector Descendiente

Se escribe con un espacio entre el padre y el hijo. Solo aplica al hijo que esté dentro del padre.

Ejemplo: .lista-especial li { color: purple; }

  • Soy afectado por el selector .lista-especial li
  • Yo también estoy dentro de .lista-especial
  • Los tres recibimos el estilo especial

Esta lista no tiene la clase, sus li no reciben el estilo especial:

  • No tengo la clase lista-especial
  • El selector descendiente no me afecta

5. Selector Múltiple (coma)

Se separan con coma (,). Aplica el mismo estilo a todos sin repetir código.

Ejemplo: h1, h2, h3 { font-family: Arial; }

Este h3 comparte estilo con el h4 y el párrafo de abajo

Este h4 también recibe el mismo estilo

Este párrafo también está incluido en el selector múltiple.
