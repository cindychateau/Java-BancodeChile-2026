Box Model

En este ejercicio aprenderemos sobre el modelo de caja: margin, padding, border, color y background-color.

¿Qué es el Box Model?

En CSS, cada elemento HTML es una caja con 4 capas (de afuera hacia adentro):

MARGIN - Espacio exterior
BORDER - Borde
PADDING - Espacio interior
CONTENIDO / CONTENT

Colores: color y background-color

color cambia el texto. background-color cambia el fondo.

¡Hola! Soy el primer div de color light sea green.
¡Hola! Soy el segundo div de color fucsia.
¡Hola! Soy el tercer div de color azul marino.

Padding: espacio interior

El padding es el espacio entre el contenido y el borde. Está DENTRO del elemento.

Esta caja NO tiene padding. El texto toca el borde.
Esta caja SÍ tiene padding. El texto tiene espacio.

Margin: espacio exterior

El margin es el espacio FUERA del elemento. Separa una caja de las que la rodean.

Caja sin margin extra.
Esta caja tiene margin-top: 30px.
Esta caja tiene margin-left: 50px.

Border: el borde de la caja

El border tiene tres partes: grosor, tipo y color.

border: 3px solid (sólido)
border: 3px dashed (rayado)
border: 3px dotted (puntitos)
