## 📋 Información importante

### Actividades CORE
Las actividades CORE son **obligatorias** para el módulo. Para acceder al examen final, deberán completar al menos el **90%** de estas.

### Asistencia a clases
La asistencia a clases con la instructora principal es **obligatoria**. Asimismo, deberán participar en los talleres prácticos, los cuales forman parte integral del proceso formativo.
En caso de presentar alguna dificultad o compromiso que les impida asistir, les solicitamos informar con anticipación.

### Modalidad de clases
Las clases se realizarán a través de **Zoom**. Les recomendamos descargar e instalar la aplicación con anticipación para evitar inconvenientes al momento de conectarse.

## 📚 Sobre el curso

Este curso proporciona una introducción completa a Java, cubriendo desde la orientación básica hasta la programación orientada a objetos, y desarrollo de aplicaciones web con Spring. El curso está diseñado para equipar a los estudiantes con las habilidades necesarias para desarrollar aplicaciones Java eficientes y seguras, utilizando Java Spring para la implementación​​.
